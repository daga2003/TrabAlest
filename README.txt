COMO COMPILAR E EXECUTAR (VIA LINHA DE COMANDO):

1. Abra o terminal na raiz deste projeto (onde a pasta 'src' está localizada).

2. Compile todos os arquivos Java:
   javac -d bin src/br/pucrs/alest1/*.java src/br/pucrs/alest1/colecoes/*.java src/br/pucrs/alest1/torneio/*.java

3. Execute a aplicação:
   java -cp bin br.pucrs.alest1.Main

REQUISITOS DE AMBIENTE:
- JDK 11 ou superior instalado e configurado no PATH do sistema.

ESTRUTURA DO PROJETO:
- src/      : Contém o código-fonte dividido em pacotes organizados.
- bin/      : Pasta gerada automaticamente após a compilação (arquivos .class).
- README.txt: Este arquivo de instruções.
